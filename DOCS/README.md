# 文档入口

本目录用于保存项目需求、设计和使用说明。当前项目需求以
`DOCS/LatestRequirementDocument.md` 为唯一权威来源。

## 当前有效文档

- `LatestRequirementDocument.md`：Info Collector NG v2.2 概要设计手册。
- `ImageOcrCollectionRequirementUpdate.md`：图片内嵌数据采集能力需求更新草案，描述本地轻量 OCR 与 Rule v2 的后续扩展方向。
- `manual.md`：旧版使用手册，仍可作为历史操作参考；涉及架构目标时以最新需求文档为准。

## 文档清理规则

- 旧版需求文档、早期看板草案和与 v2.2 冲突的 Crawl4AI / AI 集成方案不再保留。
- 新增需求或设计文档时，必须在本文登记有效状态。
- 涉及项目目标、架构边界、AI / Agent / Crawl4AI 使用策略的判断，均以
  `LatestRequirementDocument.md` 为准。
